## Q3

Sean: $A = {-2,-1,0,1,2,3}; B = {2,3,4,5,6};C = {1,3,5,7}$

1. $A∪B = {-2,-1,0,1,4,5,6}$
2. $A-C = {-2,-1,0,2}$
3. $C-A = ∅$
4. $A∩B = {2,3}$
5. $A∩B∩C = {3}$
6. $(B∩C) - A = ∅$
7. $B - (A∪C) = {4,6}$

[Excalidraw Solution 3](./main.excalidraw)

- $B = 23$
- $A = 17$
- $C = 25$
- $A \cap B \cap C = 2$
- $A \cup B = 4$
- $C \cup B = 6$
- $A \cup C = 3$
